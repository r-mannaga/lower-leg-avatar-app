# lower-leg-avatar-app
卒業研究で開発した下腿アバター表示アプリ

## 概要
座位で膝を曲げた状態で行う片足の上下運動に合わせ、下腿アバターを連動表示させるアプリです。

## 背景・目的
- ユーザの足の上下運動とアバターの動きを同期させることで、身体所有感の変化を評価することを目的としています。
- 身体所有感を客観的に評価するため、脅威物体3種とボールが落ちる映像を提示します。

## 主な機能
- カメラで黄緑色マーカーを検出し，下腿アバターを表示
- マーカーの上下動に応じてアバターを連動
- 動作1回ごとに音を鳴らし，最大10回までカウント
- スペースキー入力により，あらかじめ設定した順序で脅威物体が接触する映像を提示
- 脅威物体出現時にArduinoへ信号を送信

## 実行方法
1. Processingを起動
2. `lower_leg_avatar_app.pde` を開く
3. 実行（Runボタン）
4. 1度画面内をクリックし、黄緑色マーカーをカメラに映す

## データについて
実行に必要なデータは `data` フォルダに含まれています。

## 注意事項
本アプリは大画面（フルHD以上）を前提に設計されています。  
ノートPCなどではレイアウトが崩れます。
描画処理の負荷を軽減するため，使用した画像はあらかじめ大きいサイズで用意しており、
実行時にスケーリング処理を行わない設計としています。

## 工夫した点
レーザー血流計で指先の血流を計測する際に、映像再生のタイミングを正確に把握できるよう工夫しました。

具体的には、アプリ上で映像を再生したタイミングでArduinoへ信号を送信し、
その信号をオシロスコープに入力することで，血流データと映像再生タイミングの両方を同期して記録できるようにしました。

これにより、どの時点で刺激（映像）が与えられたかを明確にし、計測データの分析精度を向上させています。
